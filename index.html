We're currently making Lasertag

Lasertag (one word) is a mixed reality game that turns any space into a laser tag arena for you and your friends.

The game scans your room in real time so virtual bullets collide with real-world walls. Headsets connect to each other automatically, making setup super easy.

Get the game! ➔

Join our Discord ➔

GitHub repo ➔


Right now we're focusing on core features, so the game is pretty bare bones. There are also plenty of bugs. We’re working on it!

We recommend playing on Quest 3 or 3s, as older headsets lack features important to the game.

The in-game manual has more tips for setup and gameplay.
